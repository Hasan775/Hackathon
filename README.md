There are several ways to run the project.

If your machine is x64 Windows system, run dist/app/app.exe.

If not your machine must contain python with following libraries:
1 flask
2 flask_sqlalchemy
3 flask_login
4 werkzeug
5 waitress
for this purpose you can run:
pip install flask flask_a flask_sqlalchemy flask_login werkzeug waitress

to run the server you must run:
python app.py
